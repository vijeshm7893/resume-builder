# Velicx Resume Workbench

A product by **Velicx**.
</div>

An advanced, fully client-side resume builder featuring custom templates, real-time preview, deep customization options, and seamless import/export functionality.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
